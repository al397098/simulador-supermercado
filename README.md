# Simulador de Sostenibilidad para Supermercados

Este proyecto es una herramienta web interactiva diseñada para auditar y simular el impacto ambiental y económico de las prácticas de un supermercado.

## Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3, JavaScript
* **Visualización de Datos:** Chart.js
* **Backend:** Node.js, Express (Alojado en Render)
* **Base de Datos y Autenticación:** Firebase (Firestore & Auth)
* **Despliegue:** GitHub Pages
* 
## Funcionalidades Principales
* Autenticación segura mediante Google OAuth.
* Cálculos de sostenibilidad en tiempo real.
* Panel de control (Dashboard) con gráficas interactivas.
* Conexión con base de datos NoSQL para almacenar auditorías.
